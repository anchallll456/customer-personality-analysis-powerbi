Interactive Power BI dashboard analyzing customer demographics, spending behavior, and marketing campaign performance to generate actionable business insights.
# 📊 Customer Personality Analysis Dashboard

## 📌 Project Summary

This Power BI project focuses on analyzing customer data to uncover meaningful insights about **customer behavior, demographics, and purchasing patterns**.

The dashboard helps businesses:

* Understand different customer segments
* Identify high-value customers
* Improve marketing strategies using data

---

## 🎯 Problem Statement

Businesses often struggle to:

* Identify target customer groups
* Understand purchasing behavior
* Measure campaign effectiveness

This dashboard solves these problems using data visualization.

---

## 📁 Dataset Details

* **Source:** Customer Personality dataset
* **Format:** CSV
* Includes:

  * Age, income, education
  * Spending across product categories
  * Campaign responses
  * Customer engagement metrics

---

## 📊 Key Dashboard Features

* 📈 **Spending Analysis by Category**
* 👥 **Customer Segmentation (Income, Age Group)**
* 🎯 **Marketing Campaign Performance**
* 🛍️ **Product Preference Insights**
* 📅 **Trend Analysis**

---

## 🛠️ Tools Used

* Power BI
* Data Cleaning & Transformation
* DAX (Data Analysis Expressions)
* CSV Dataset

---

## 🚀 How to Use

1. Download the `.pbix` file from the **Reports/** folder
2. Open it in Power BI Desktop
3. Interact with filters and visuals

---

## 📌 Insights Gained

* High-income customers spend more on premium products
* Certain campaigns have higher conversion rates
* Customer segmentation helps target marketing effectively
